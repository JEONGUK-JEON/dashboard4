# 범우연합 채권현황 대시보드

2026년 4월 기준 법인별 채권 현황을 시각화하는 정적 웹 대시보드입니다.

## 파일 구성

```
bond-dashboard/
├── index.html    # 메인 HTML (비밀번호 게이트 포함)
├── styles.css    # 스타일시트
├── script.js     # 차트·인터랙션 로직 (Chart.js 4.4.1)
├── data.json     # 채권 데이터 (집계시트 기반, 단위: 억원)
└── README.md     # 이 파일
```

## 기능

| 항목 | 설명 |
|------|------|
| **비밀번호 게이트** | SHA-256 해시 기반 인증 (기존 매출 대시보드와 동일) |
| **법인 선택** | 범우연합 합계 / 국내사·해외사·판매사 그룹 / 개별 19개 법인 |
| **KPI 카드** | 채권합계, A·B·C·D등급별 잔액 및 비중, 전월대비 증감 |
| **월별 추이** | 5개월 스택바 + 합계 꺾은선 (2025-12 ~ 2026-04) |
| **등급별 비중** | 도넛 차트 (당월 기준) |
| **상세 테이블** | 5개월 × 5개 항목 |
| **법인별 비교** | 그룹 내 법인 채권 수평 비교 |
| **전월대비 증감** | 양수(적색)/음수(청색) 막대 |

## 채권 등급 기준

| 등급 | 기간 | 색상 |
|------|------|------|
| **A** | 3개월 이내 | 파랑 |
| **B** | 4~6개월 | 민트 |
| **C** | 7~12개월 | 황색 |
| **D** | 1년 초과 | 빨강 |

## GitHub Pages 배포

저장소 루트에 파일을 넣고 Settings → Pages → `main` 브랜치로 활성화.

## 비밀번호 변경

`index.html` 상단 `window.DASHBOARD_CONFIG.passwordHash` 값을 원하는 비밀번호의 SHA-256 hex로 교체.

## 데이터 갱신

`data.json`의 `entities` 배열을 수정하거나 교체하면 자동으로 반영됩니다.
월이 추가될 경우 `script.js`의 `PERIODS` / `PERIOD_LABELS` 배열에 추가하고
`data.json` 각 법인의 `periods` 객체에 새 월 데이터를 추가하세요.
