# GitHub Pages 배포용 매출 분석 대시보드

## 포함 파일
- `index.html`
- `styles.css`
- `script.js`
- `data.json`

## 이번 수정사항
- 법인 선택 버튼을 좌측 스크롤형이 아니라 상단 버튼형으로 변경
- 연합 총계 버튼 추가
- 배경을 흰색 계열로 재디자인
- 월별매출 추이 그래프를 흰 배경용 스타일로 변경
- KPI는 아래 6개로 고정
  - 당월매출
  - 전년동월대비
  - 누계
  - 작년대비누계
  - 작년 총매출
  - 12개월 평균
- 비밀번호는 SHA-256 해시 방식으로 유지
- 비밀번호는 `buhmwoo2026`

## 배포 방법
1. 이 압축을 해제합니다.
2. GitHub 저장소를 생성하거나 기존 저장소를 엽니다.
3. 위 파일을 저장소 루트에 업로드합니다.
4. GitHub 저장소의 `Settings > Pages`에서 배포 소스를 `main` 브랜치의 `/ (root)`로 지정합니다.
5. 배포 후 `https://사용자명.github.io/저장소명/`으로 접속합니다.

## 월간 업데이트
- `data.json`의 월별 매출 데이터만 교체하면 화면이 자동 반영됩니다.
- 기준월은 `data.json`의 `latestMonth`를 따릅니다.

## 주의
- 현재 화면은 정적 웹 대시보드입니다.
- 민감한 실데이터를 공개 저장소에 그대로 업로드하는 방식은 권장하지 않습니다.
